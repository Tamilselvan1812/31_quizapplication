# Quiz Application – React

## Description
This is a simple Quiz Application built using React. Users can answer multiple-choice
questions and view their final score at the end. This project was created to practice
and demonstrate core React concepts such as state management, props, and
component-based architecture.

## Technologies Used
- React
- JavaScript (ES6)
- HTML
- CSS
- Create React App

## Features
- Multiple-choice quiz questions
- Dynamic question rendering
- Score calculation for correct answers
- Final result display
- Simple and clean UI

## React Concepts Used
- Functional Components
- Props
- State management using useState
- Conditional Rendering
- Component reusability

## How to Run the Project

1. Install dependencies  
npm install

2. Start the application  
npm start

The application will run on:  
http://localhost:3000

## How the Application Works
- Quiz questions are stored as data inside the application.
- The current question index and score are managed using React state.
- When a user selects an answer, the state updates and the next question is rendered.
- After the last question, the final score is displayed.
